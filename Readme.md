# 🎧 Spotify Web Player Clone

A responsive **Spotify-inspired web music player UI** built using **HTML & CSS**. This project replicates the layout and design of Spotify’s web player, including sidebar navigation, playlists, cards, and a music control bar.

---

## 🚀 Features

* 🎵 Sidebar with navigation (Home, Search, Library)
* 📂 Playlist & podcast sections
* 📊 Cards layout for:

  * Recently Played
  * Trending Near You
  * Featured Charts
* 🎛️ Sticky navigation bar
* 🎶 Music player UI (playback controls, progress bar, volume)
* 📱 Responsive design (basic media queries included)
* 🎨 Clean and modern UI using Flexbox

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **Font Awesome (Icons)**
* **Google Fonts (Montserrat)**
